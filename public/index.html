DBA Panel — GRANT & REVOKE

Simulasi pemberian dan pencabutan hak akses user database. Konsep ini adalah inti dari tugas seorang Database Administrator.

GRANT

Berikan Akses

Memberikan privilege kepada user agar bisa mengakses tabel tertentu.

REVOKE

Cabut Akses

Mencabut privilege dari user sehingga tidak bisa mengakses tabel lagi.
